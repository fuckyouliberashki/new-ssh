**⚠️ ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ**
**Скрипт для быстрого создания временного пользователя с SSH-доступом по паролю**

**⚠️ ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ**

Этот скрипт **крайне опасно** хранить и распространять в публичном доступе.  
Любой, кто получит доступ к запущенному скрипту или его копии, сможет создать себе учётную запись на вашем сервере.

Поэтому:

- **Никогда** не коммитьте сам скрипт в публичный репозиторий
- Используйте его только локально или в приватном репозитории
- После использования — удалите или заархивируйте репозиторий
- Лучший вариант — вообще не хранить такой код в git

## Что делает скрипт

- Генерирует случайное имя пользователя (11–12 символов)
- Генерирует случайный сложный пароль (15–16 символов)
- Создаёт нового системного пользователя
- Устанавливает сгенерированный пароль
- (опционально) добавляет пользователя в группу `sudo` / `wheel`
- Выводит готовую команду для подключения по SSH

## Требования

- Linux сервер (Debian / Ubuntu / CentOS / AlmaLinux / Rocky и др.)
- Запуск от имени **root** или с `sudo`
- Установленный пакет `whois` (для `mkpasswd`) — опционально, но рекомендуется

```bash
# Ubuntu / Debian
sudo apt update && sudo apt install -y whois

# CentOS / Alma / Rocky / Fedora
sudo dnf install -y whois    или    sudo yum install -y whois
